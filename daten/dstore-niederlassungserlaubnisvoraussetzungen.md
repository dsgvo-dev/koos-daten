# Niederlassungserlaubnisvoraussetzungen

## Definition

## Felder

- Aufenthaltsdauer
- Lebensunterhaltssicherung
- Wohnraum
- Sprachkenntnisse
- Rentenbeiträge
- Straffreiheit
